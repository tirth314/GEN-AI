# Generative AI Repository

This repository contains my work, experiments, and learning journey in Generative Artificial Intelligence (GenAI).

The goal of this project is to explore how modern AI systems generate text, images, code, and other forms of content using machine learning and deep learning techniques. It includes implementations, notes, projects, and practical examples related to Large Language Models (LLMs), prompt engineering, and AI application development.

## What You'll Find Here

* Fundamentals of Generative AI
* Prompt Engineering techniques
* Large Language Model (LLM) applications
* Machine Learning and Deep Learning projects
* AI-powered tools and automation workflows
* Experiments, notebooks, and project documentation

## Technologies

* Python
* NumPy
* Pandas
* Scikit-learn
* TensorFlow 
* Anthropic/OPENAI APIs
* Jupyter Notebook

## Objective

This repository serves as a collection of my learning resources, hands-on projects, and research in Generative AI. The focus is on understanding the underlying concepts, implementing practical solutions, and building applications that leverage modern AI technologies.

## Future Work

As I continue learning, this repository will be updated with new projects, model implementations, experiments, and insights from the rapidly evolving field of Generative AI.
